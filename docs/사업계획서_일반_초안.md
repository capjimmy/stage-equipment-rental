## 2. 문제 인식 (Problem)

### 2.1 시장의 문제점

#### 공급자 측면 (장비 보유업체)
| 문제점 | 현황 |
|--------|------|
| 낮은 자산 활용률 | 보유 장비의 평균 가동률 30% 미만 |
| 영업/마케팅 역량 부족 | 개별 홍보 한계로 신규 고객 확보 어려움 |
| 재고 관리의 어려움 | 수기/엑셀 기반 관리로 이중예약 발생 |
| 대금 회수 리스크 | 개인 거래로 인한 미수금 발생 |

#### 수요자 측면 (공연/연극/영상 제작사)
| 문제점 | 현황 |
|--------|------|
| 정보 분산 | 장비별로 개별 업체 탐색 필요 |
| 가격 불투명 | 동일 장비 가격 편차 2~3배 |
| 가용성 확인 어려움 | 전화/문자로 일일이 확인 필요 |
| 품질 불확실성 | 장비 상태 사전 확인 불가 |

### 2.2 목표 시장 분석

#### TAM (전체 시장)
- 국내 공연예술 시장 규모: **1.2조원** (2024년 기준)
- 연평균 성장률: 8.5%

#### SAM (유효 시장)
- 무대장비 렌탈 시장: **2,400억원**
- 의상/소품/무대장치 포함

#### SOM (목표 시장)
- 온라인 플랫폼 전환 가능 시장: **480억원** (20%)
- 초기 목표 (3년): **24억원** (5% 점유)

### 2.3 경쟁사 분석

| 구분 | 기존 렌탈업체 | 종합 렌탈 플랫폼 | 당사 |
|------|-------------|----------------|------|
| 전문성 | 높음 | 낮음 | 높음 |
| 접근성 | 낮음 | 높음 | 높음 |
| 가격 투명성 | 낮음 | 높음 | 높음 |

## 3. 해결책 (Solution)

### 3.1 서비스 개요

**"Stage Rental" - 무대장비 통합 대여 플랫폼**

공연/연극/뮤지컬/영상 제작에 필요한 의상, 소품, 무대장치, 음향/조명 장비를
한 곳에서 검색하고 예약할 수 있는 B2B 온라인 플랫폼

### 3.2 핵심 기능

#### 수요자(렌터) 기능
| 기능 | 설명 |
|------|------|
| 통합 검색 | 날짜/카테고리/태그 기반 장비 검색 |
| 실시간 가용성 확인 | 캘린더 기반 예약 가능 여부 즉시 확인 |
| 장바구니 & 일괄 주문 | 여러 장비 한번에 대여 신청 |
| 배송/픽업 선택 | 수령 방식 자유 선택 |
| 주문 이력 관리 | 과거 대여 이력 조회 및 재주문 |

#### 공급자(업체) 기능
| 기능 | 설명 |
|------|------|
| 상품 등록 관리 | 장비 정보, 사진, 가격 등록 |
| 자산 개별 관리 | 시리얼번호 기반 개별 자산 추적 |
| 예약 일정 관리 | 캘린더 기반 예약 현황 확인 |
| 차단 기간 설정 | 정비/점검 기간 예약 차단 |
| 정산 관리 | 월별 정산 내역 확인 |

#### 관리자 기능
| 기능 | 설명 |
|------|------|
| 주문 상태 관리 | 요청→승인→발송→회수 워크플로우 |
| 입금 확인 | 결제 상태 관리 |
| 통계 대시보드 | 매출/주문/사용자 통계 |
| 사용자 관리 | 업체/고객 계정 관리 |

## 4. 성장 전략 (Scale-up)

### 4.1 비즈니스 모델

#### 수익 구조
| 수익원 | 설명 | 예상 비중 |
|--------|------|----------|
| 거래 수수료 | 거래액의 10~15% | 70% |
| 광고/프로모션 | 상위 노출, 배너 광고 | 15% |
| 프리미엄 서비스 | 업체 대시보드 고급 기능 | 10% |
| 보험 연계 | 장비 손상 보험 중개 | 5% |

#### 가격 정책
- 기본 수수료: 거래액의 10%
- 프리미엄 업체: 월 99,000원 (수수료 7%)

### 4.2 마케팅 전략

#### Phase 1: 시장 진입 (0~6개월)
- 공연예술 단체 직접 영업
- 대학 연극/뮤지컬 동아리 타겟
- 인플루언서 협업 (연극배우, 스태프)

#### Phase 2: 성장 (6~12개월)
- SEO/SEM 마케팅
- 공연예술 전문 매체 제휴
- 전시회/박람회 참가

#### Phase 3: 확장 (12~24개월)
- 타 지역 확대 (부산, 대구 등)
- 영상 제작사 시장 진출
- B2B 파트너십 확대
