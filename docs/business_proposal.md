## Executive Summary

### 스테이지박스란?

**스테이지박스**는 공연/행사 의상을 **온라인으로 검색하고, 장바구니에 담아 대여**하는 B2B 플랫폼입니다.

### 두 가지 대여 방식

| 방식 | 설명 | 적합한 경우 |
|------|------|------------|
| 🛒 **장바구니 대여** | 개별 의상을 자유롭게 선택, 믹스 매치 | 창작극, 학교 공연, 기업 행사 |
| 📦 **풀세트 대여** | 공연별 전체 의상 패키지 (30% 할인) | 라이선스 공연, 대형 프로덕션 |

### 제안 요약

| 항목 | 내용 |
|------|------|
| **비즈니스 모델** | 귀사 의상 위탁 → 스테이지박스 등록 → 대여 수익 창출 |
| **수익 배분** | **50% : 50%** (귀사 : 스테이지박스) |
| **초기 비용** | 없음 (촬영, 등록, 마케팅 전담) |
| **대여 고객** | 뮤지컬/연극 제작사, 학교, 기업행사, 지역문화재단 |

### 핵심 수치

| 지표 | 수치 |
|------|------|
| 2024년 공연예술시장 규모 | **1.5조원** |
| 연간 공연 작품 수 | **30,273건** |
| 공연 후 유휴화되는 의상 | **90%+** |
| 전문 B2B 의상 대여 플랫폼 | **0개** |

## 1. 서비스 모델

### 🛒 장바구니 대여

**"내가 직접 골라서 구성"**

- 개별 의상을 자유롭게 선택
- 다양한 작품에서 믹스 매치
- 수량 자유롭게 조절
- 예산에 맞춰 구성

**적합한 경우:** 창작극, 학교 공연, 기업 행사, 저예산 프로젝트

### 📦 풀세트 대여

**"검증된 구성 그대로"**

- 공연별 전체 의상 패키지
- 캐릭터별 세트 구성
- 주연~앙상블 전체 포함
- **30% 패키지 할인**

**적합한 경우:** 라이선스 공연, 대형 프로덕션, 재공연

### 장바구니 시스템 예시

```
🛒 내 장바구니 (3개 의상 선택됨)
─────────────────────────────────────────
☑️ [나폴레옹] 황제 대관복
   주연급 | 1주 대여                    400,000원

☑️ [나폴레옹] 조제핀 황후 드레스
   주연급 | 1주 대여                    400,000원

☑️ [나폴레옹] 귀족 드레스 5벌 세트
   앙상블 | 1주 대여                    600,000원
─────────────────────────────────────────
                              합계  1,400,000원
```

### 풀세트 패키지 예시

| 패키지 | 구성 | 개별 대여 시 | 패키지 가격 | 할인 |
|--------|------|-------------|------------|------|
| 나폴레옹 풀세트 | 주연 10벌 + 앙상블 22벌 | 12,000,000원 | **8,400,000원** | 30% |
| 바람과 함께 사라지다 | 주연 8벌 + 앙상블 36벌 | 15,000,000원 | **10,500,000원** | 30% |
| 오!캐롤 풀세트 | 주연 6벌 + 앙상블 13벌 | 6,500,000원 | **4,550,000원** | 30% |

|------|----------|----------|------------|
| 🎭 **대형 제작사** | 빠른 의상 확보, 품질 보증 | 풀세트 | 500만~2,000만원 |
| 🎭 **중소 극단** | 비용 절감, 유연한 구성 | 장바구니+풀세트 | 100만~500만원 |

### 의상 유휴화 문제

> **"보관해도 손해, 버려도 손해"**
>
> 수백만~수천만 원을 투자해 제작한 의상이 공연 종료 후 창고에서 방치되거나 폐기

- 🔴 공연 후 재활용 안 되는 의상: **90%+**
- 🔴 대형 제작사 연간 창고 비용: **수천만원**

## 6. 플랫폼 기능

### 고객 이용 흐름

```
🔍 검색 → 🛒 장바구니 → 📋 견적 → 💳 결제 → 📦 수령
```

### 핵심 기능

| 기능 | 설명 |
|------|------|
| **스마트 검색** | 작품/캐릭터/시대/장르/가격대별 검색 |
| **장바구니** | 자유 구성, 수량 조절, 실시간 합계, 저장/공유 |
| **풀세트 패키지** | 원클릭 선택, 30% 자동 할인, 커스텀 가능 |
| **대여 관리** | 예약 캘린더, 반납 알림, 이력 조회 |

### 의상 상세 정보

- 고해상도 사진 (전면, 후면, 상세)
- 작품 정보 (뮤지컬명, 캐릭터, 장면)
- 스펙 (사이즈, 소재, 구성품)
- 상태 (컨디션 등급 A/B/C)
- 실시간 예약 현황
