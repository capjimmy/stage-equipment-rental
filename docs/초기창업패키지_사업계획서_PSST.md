## 2. 창업아이템 개요 (요약)

### 2.1 한 줄 소개
> 공연/연극/뮤지컬 제작에 필요한 의상, 소품, 무대장치를 한 곳에서 검색하고 실시간으로 예약할 수 있는 **B2B 무대장비 통합 대여 플랫폼**

### 2.2 아이템 개요

| 구분 | 내용 |
|------|------|
| **서비스명** | Stage Rental (스테이지 렌탈) |
| **서비스 유형** | B2B 온라인 플랫폼 |
| **목표 시장** | 공연예술 (연극, 뮤지컬, 콘서트) / 영상 제작 |
| **핵심 가치** | 분산된 무대장비 정보의 통합, 실시간 가용성 확인, 거래 효율화 |
| **수익 모델** | 거래 수수료 (10~15%), 프리미엄 구독, 광고 |
| **차별성** | 무대장비 특화 / 자산별 개별 관리 / 차단 기간 설정 |

### 2.3 개발 현황

| 항목 | 상태 | 설명 |
|------|------|------|
| 비즈니스 모델 검증 | 완료 | 장비 업체 5개사 인터뷰 완료 |
| MVP 개발 | 95% 완료 | 웹 플랫폼 개발 완료, 베타테스트 중 |
| 기술 스택 | 확정 | Next.js + NestJS + TypeORM |

## 3. 문제인식 (Problem) - 배점 높음

### 3.1 창업 배경 및 동기

#### 개인적 경험에서 발견한 문제
```
[예시 - 실제 경험으로 수정 필요]
대학 연극 동아리 활동 중 공연 의상과 소품을 구하기 위해
10개 이상의 업체에 개별 연락해야 했던 경험이 있습니다.
업체마다 가격이 달랐고, 가용 여부 확인에만 일주일이 소요되었습니다.
이 비효율을 해결하고자 플랫폼 창업을 결심했습니다.
```

### 3.2 시장의 문제점 (Pain Point)

#### 3.2.1 수요자(렌터) 측면

| 문제점 | 현황 | 근거 자료 |
|--------|------|----------|
| **정보 분산** | 장비 업체 정보가 분산되어 개별 탐색 필요 | 업체 인터뷰 결과, 평균 5~10개 업체 개별 연락 |
| **가격 불투명** | 동일 장비 가격 편차 2~3배 | 동일 군복 의상 일대여료 3만원~8만원 |
| **가용성 확인 어려움** | 전화/문자로 일일이 확인 필요 | 응답까지 평균 1~3일 소요 |
| **품질 불확실성** | 장비 상태 사전 확인 불가 | 수령 후 상태 불량 발견 사례 다수 |

#### 3.2.2 공급자(업체) 측면

| 문제점 | 현황 | 근거 자료 |
|--------|------|----------|
| **낮은 자산 활용률** | 보유 장비 평균 가동률 30% 미만 | 업체 인터뷰 (5개사 평균) |
| **영업 역량 부족** | 개별 홍보 한계로 신규 고객 확보 어려움 | 신규 고객 80%가 기존 고객 소개 |
| **재고 관리 어려움** | 수기/엑셀 기반 관리로 이중예약 발생 | 월 1~2건 이중예약 발생 |
| **대금 회수 리스크** | 개인 거래로 인한 미수금 | 연간 미수금 비율 5~10% |
